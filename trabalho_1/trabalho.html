Trabalho 1 - Desenhando uma figura

Teclas ativas no desenho ao lado

- Tecla page up move a o foguete para trás (aumenta o Z)

- Tecla page down move a o foguete para frente (diminui o Z)

- Tecla A move a o foguete para esquerda (diminui o X)

- Tecla D move a o foguete para direita (aumenta o X)

- Tecla S move a o foguete para baixo (diminui o Y)

- Tecla W move a o foguete para cima (aumenta o Y)
